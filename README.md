# Weekplanner

Een drag & drop weekplanner voor freelancers. Plan je taken per klant over de week, met capaciteitsoverzicht per dag.

## Features
- Taken toevoegen met klant, uren en prioriteit
- Drag & drop naar weekdagen
- Capaciteitsmeter per dag
- Weekoverzicht per klant
- Lokale opslag (localStorage)

## Tech Stack
- React 18 + Vite
- Klaar voor Vercel deployment
- Later: Supabase voor persistente opslag

## Development
```bash
npm install
npm run dev
```

## Deploy
Push naar GitHub en koppel aan Vercel. Werkt out of the box.
